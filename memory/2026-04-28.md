# 2026-04-28 — BrainSAIT Training Hub Overhaul

## Summary
Full debugging and rebuilding of the BrainSAIT.org training navigation. Started with the Training header link returning 404/blank page, ended with a fully working unified training hub (React SPA in `hub/`) and enriched course content.

## Key Events

### Navigation/Routing Fixes
- Homepage Training link originally pointed to `/training/` which got intercepted by Next.js client-side router → rendered blank
- Cloudflare Worker `brainsait-app-router` was returning hard 404s for `/training/*` — added pass-through to Pages project
- `_redirects` had infinite redirect loops due to splat patterns matching themselves
- Final routing: `/training/` → `_redirects` 200 rewrite → serves `training/index.html` (the hub React SPA)
- Worker routes: `brainsait.org/training*` and `brainsait.org/incubator*`

### The Blank Page Bug (real root cause)
- `training/index.html` and `training/home.html` were empty shells with `<div id="root">` pointing to stale JS bundle from `/training/courses/assets/`
- That bundle was the Udacity-style catalog JS, not the old hub
- **Fix:** Found real hub source in `hub/` (Vite React app), built with `--base=/training/`, copied output into `training/`

### Course Content Enhancement
- Hub source in `hub/` — React app with `CatalogView`, `CourseDetailView`, hash routing, i18n (EN/AR), filter by source/category
- Enhanced descriptions: Collective Brainpower, NPHIES-AI Mastery, FHIR R4 Implementation
- Added new course: **Healthcare AI Launchpad** (2-week bootcamp)
- Added chip badges: Leadership, Live, Hands-on
- 20 total courses in dataset
- Learning paths: AI Microdegree, BrainSAIT Certificate, Healthcare Data Scientist, RCM Acceleration
