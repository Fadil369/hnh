- Cloudflare Workers have 1MB code size limit (we're at ~191KB uncompressed, fine)

- ElevenLabs TTS + AI in sequence can exceed CF Worker 30s limit ‚Äî keep AI tokens ‚â§250 & TTS text ‚â§400 chars for voice chat
